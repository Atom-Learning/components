# CSSWrapper

If `CSSWrapper` is passed a truthy `css` prop, it renders its children wrapped in a `Box` that has that `css`. If `css` is falsy, `CSSWrapper`
just renders its children.

Use `CSSWrapper` to wrap other components that have no natural container. For example, our `InputField` component renders something like this:

```tsx
<CSSWrapper css={css}>
  <Label htmlFor={name} css={{ mb: '$2' }}>
    {label}
  </Label>
  <Input name={name} id={name} {...props} />
</CSSWrapper>
```
