# Panel

The `Panel` is a specialised instance of `Box`. It has rounded corners, a box shadow and padding.

Use a `Panel` to organise content into a block.

  <Story name="Basic">
    <Panel size="md">
      <p>Wow, the eye is really drawn to this content huh?</p>
    </Panel>
  </Story>
