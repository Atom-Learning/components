# TextField

The `TextField` component combines an `Input` and `label` in a standardised way, linking them and styling both. Its `css` prop applies styling to a containing `Box`,
making it easy to size the `TextField` and position it relative to other components.

      name="email"
      css={{ width: '250px', mb: '$3' }}
    />
    <TextField
      label="Mobile number"
      name="mobile"
      type="number"

    />
