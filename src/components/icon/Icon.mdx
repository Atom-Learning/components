`Icon` requires an SVG from our icon library and adds standardised `size` options. Review available icons on the [icon listing page](/theme/icons)

## Sizes

```tsx live
<Icon is={Danger} size="sm" />
```

```tsx preview
<Icon is={Eye} size="md" />
```

```tsx preview
<Icon is={Brightness} size="lg" />
```
