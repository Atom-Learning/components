`Icon` effectively takes an SVG from our icon library and adds standardised `size` options.

Make sure any SVGs added to the library have been optimised, for example with [SVGOMG](https://jakearchibald.github.io/svgomg/).

## Sizes

```jsx live center
<Icon is={Check} size="sm" />
```

```jsx preview
<Icon is={Check} size="md" />
```

```jsx preview
<Icon is={Check} size="lg" />
```
