`Icon` requires an SVG from our icon library and adds standardised `size` options.

## Sizes

```jsx live
<Icon is={Danger} size="sm" />
```

```jsx preview
<Icon is={Eye} size="md" />
```

```jsx preview
<Icon is={Brightness} size="lg" />
```
