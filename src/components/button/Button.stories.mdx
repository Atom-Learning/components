# Button

The `Button` component is a very thin wrapper around an HTML `button` element. It adds default styling and the `css` prop. By default `Primary` theme is displayed.

<Preview>
  <Story name="Basic">
    <Button css={{ mb: '$2' }}>Primary</Button>
  </Story>
</Preview>

## Variants

These are the available `themes` for the `Button` component: `Primary` (default), `Secondary`, `Tertiary`, `Success`, `Warning`, `Danger`
