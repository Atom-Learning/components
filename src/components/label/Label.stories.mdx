# Label

`Label` is a thin wrapper around the basic HTML `<label/>` element. It adds default styling and the `css` prop. Use it to compose more complex field components and form related component.

## Accessibility

For accessibility purposes the component needs either to have the `htmlFor` prop set to link it to the control it is associated with or to wrap around the component that it is labelling

## Sizes

The default size is `md`.
