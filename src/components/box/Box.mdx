title: Box
component: Box
description: Box is our most fundamental layout component.
---

Many other components are composed of `Box`, for example, this card:

```jsx live center
<Box
  as="article"
  css={{
    border: '1px solid $tonal300',
    borderRadius: '$1',
    overflow: 'hidden'
  }}
>
  <Box as="header" css={{ bg: '$tonal300', p: '$3' }}>
    <Heading as="h2" size="sm">
      Cats!
    </Heading>
  </Box>
  <Box as="figure" css={{ m: 0, p: '$3' }}>
    <Image
      src="http://placekitten.com/280/120"
      alt="Cats having a nice time"
      css={{ mb: '$2' }}
    />
    <Text as="figcaption" size="sm">
      Here is a cute cat
    </Text>
  </Box>
  <Box as="footer" css={{ p: '$3' }}>
    <Text>Remember: cats have whiskers!</Text>
  </Box>
</Box>
```

`Box` falls back to rendering a `div`, but we can use it to render any other container element with e.g. `<Box as="section">`. For example, a card component will use `Box` to render a `section`, `header` and `footer`.

Avoid rendering any of these elements directly in our components â€” `Box` has all the same flexibility, but with the addition of the powerful `css` prop and various styling utilities based on our themes.

```jsx live center
<Box
  css={{
    bg: '$tertiary500',
    borderRadius: '$1',
    boxShadow: '$1',
    color: 'white',
    p: '$4',
    textAlign: 'center',
    width: 250
  }}
>
  Some text in a Box
</Box>
```
