---
title: Popover
component: Popover
description: Popover provides a styled actionable popover.
---

The `Popover` component needs a `visible` boolean prop that determins whether it will show the popover or not.

```jsx live center
<Popover visible aria-label="Password hints" />
<Popover visible={false} aria-label="Password hints" />
```

It can display a `content` and a `child` as a trigger

```jsx live center
<Popover visible aria-label="Password hints" content="Here is some content" />
<Popover aria-label="Password hints" content="Here is some content">
  <Button>Click me</Button>
</Popover>
```

## Accessibility

For accessibility purposes the `aria-label` is mandatory
