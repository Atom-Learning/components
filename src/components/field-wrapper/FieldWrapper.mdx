---
title: Field Wrapper
component: FieldWrapper
description: A utility component to help with composing consistent form fields
category: Forms
---

`FieldWrapper` renders a `Label` and (when applicable) a `ValidationError`. Use it to wrap any type of input to create a consistent set of form fields.

`InlineFieldWrapper` does the same but renders the label inline with the input. Use this for `Checkbox`es and `Radio`s.

```jsx preview
<FieldWrapper label="Example Field" fieldId="example">
  <Input name="example" id="example" />
</FieldWrapper>

<InlineFieldWrapper label="Example Checkbox" fieldId="example2">
  <Checkbox name="example2" id="example2" />
</FieldWrapper>
```
