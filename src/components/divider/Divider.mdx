---
title: Divider
component: Divider
description: Use Divider to render a horizontal or vertical line which will inherit parent node's height
category: Content
---

It adds default styling and the `css` prop. Provide `orientation?` as `horizontal` or `vertical` to achieve desirable output

```tsx preview
<Divider />
```

```tsx preview
<Divider orientation="vertical"/>
```
