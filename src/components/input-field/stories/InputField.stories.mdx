# TextField

The `InputField` component combines an `Input` and `label` in a standardised way, linking them and styling both. Its `css` prop applies styling to a containing `Box`,
making it easy to size the `InputField` and position it relative to other components.

<Canvas>
  <Story name="Basic">
    <InputField
      label="Email address"
      name="email"
      css={{ width: '250px', mb: '$3' }}
    />
    <InputField
      label="Mobile number"
      name="mobile"
      type="number"
      css={{ width: '150px' }}
    />
  </Story>
</Canvas>
