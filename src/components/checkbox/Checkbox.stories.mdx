# Checkbox

`Checkbox` provides a styled checkbox. 

Checkboxes should be accompanied by labels, so rather than using `Checkbox` directly in a UI,
it's normally best to use a field component, which combines an `Checkbox` with a `Label` and
displays validation errors.
