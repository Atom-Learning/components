---
title: Tooltip
component: Tooltip
description: Tooltip provides some extra information when its child component is hovered or focussed
---

Use a `Tooltip` to make a definition or description available when a user hovers or focusses on a component.

```jsx preview
<Tooltip open>
  <Tooltip.Trigger>
    <span>Hover on me</span>
  </Tooltip.Trigger>
  <Tooltip.Content>Tooltip content</Tooltip.Content>
</Tooltip>
```
