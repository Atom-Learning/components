# Button

The `Button` component is a very thin wrapper around an HTML `button` element. It adds default styling and the `css` prop.

## Variants

There are 3 basic branded variants for the `Button` component.

- `Primary` which is the default variant
- `Secondary`
- `Tertiary`

<Preview>
  <Story name="Basic">
