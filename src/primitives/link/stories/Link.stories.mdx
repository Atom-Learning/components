# Link

The `Link` component is a very thin wrapper around an HTML `a` element. It adds default styling and the `css` prop.

<Canvas>
  <Story name="Basic">
    <Flex
      css={{
        flexDirection: 'column',
        justifyContent: 'space-between',
        width: '200px',
        height: '70px'
      }}
    >
      <Link href="https://app.atomlearning.co.uk/forgotten">
        Forgot your password?
      </Link>
      <Link
        href="https://app.atomlearning.co.uk/forgotten"
        css={{
          fontSize: '$7',
          color: '$secondary700',
          ':hover': { color: '$secondary300' }
        }}
      >
        Forgot your password?
      </Link>
    </Flex>
  </Story>
</Canvas>
